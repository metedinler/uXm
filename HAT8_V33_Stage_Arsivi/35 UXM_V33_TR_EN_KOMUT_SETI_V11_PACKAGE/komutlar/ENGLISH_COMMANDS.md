# UXM English Tool Set V11

English tools live under `tool_en/`.

| Command | Purpose |
|---|---|
| `tool_en\help.bat` | English help screen |
| `tool_en\memory_test.bat` | Run 16 MB memory-model tests |
| `tool_en\fast_scan.bat` | Scan latest result CSV for failing keys |
| `tool_en\failed_test.bat` | Rerun unique failing tests |
| `tool_en\all_test.bat` | Run all expected tests |
| `tool_en\workspace_clean.bat` | Organize workspace |
| `tool_en\report_show.bat` | Show latest report |
| `tool_en\vscode_install.bat` | Install VSCode extension |

Short options: `-k` no-build, `-D` stop-on-fail, `-n` limit, `-s` start index, `-a` name filter, `-z` timeout, `-u` apply, `-b` retire build.
