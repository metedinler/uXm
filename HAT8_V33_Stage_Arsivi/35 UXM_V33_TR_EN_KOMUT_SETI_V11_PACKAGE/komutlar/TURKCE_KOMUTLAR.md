# UXM Türkçe Komut Seti V11

| Komut | Görev |
|---|---|
| `yardim.bat` | Ana yardım ekranı |
| `derleyici_derle.bat` | `build_native.bat` ile ana derleyiciyi derler |
| `bellek_test.bat` | 16 MB esnek bellek modelini test eder |
| `hizli_tara.bat` | Son CSV sonucundan hatalı anahtarları çıkarır |
| `hatali_test.bat` | Sadece tekil hatalı testleri tekrar koşar |
| `tum_test.bat` | `uxm/tests/all_expected_known` altındaki tüm beklenenli testleri koşar |
| `alan_topla.bat` | Çalışma alanını düzenler, varsayılan deneme modudur |
| `rapor_goster.bat` | En son raporu gösterir |
| `vscode_kur.bat` | VSCode UXM dil desteğini kurar |

## Anlamlı kısa seçenekler

`-k` derleme yok, `-D` ilk hatada dur, `-n` adet, `-s` başla, `-a` ara, `-z` zaman, `-u` uygula, `-b` build emekli.
